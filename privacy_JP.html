作文くんアプリケーションのプライバシーポリシー

最終更新日：2025年7月1日

本プライバシーポリシーは、以下のアプリケーションに適用されます。

  • 作文くんSP（ドラッグ型）：英文並べ替え課題の作成と、Gmailを通じて回答を送信するアプリケーション
  • 作文くんEX（タイピング型）：英文並べ替え課題の作成と、Gmailを通じて回答を送信するアプリケーション
  • 作文くん問題作成：Google スプレッドシートとドライブを使用して課題ファイルを作成・管理するツール

Google API Services User Data Policy を含む関連規定に準拠し、ユーザーのプライバシー保護に最大限の注意を払っています。

1. 取得するデータとその利用目的

（1）作文くんSP/EX

  • Googleアカウントは本人確認（認証）にのみ使用されます。
  • Googleスプレッドシートは、課題や成績の管理に使用されます。
  • Gmailは、学習成績を担当教員に送信するために使用されます。
  • 学生のメールアドレスは送信目的のみに使用され、保存や再利用は行いません。

（2）作文くん問題作成

  • ユーザーが明示的に開いたGoogleスプレッドシートに読み書きします。
  • 課題ファイルをGoogleドライブ上に自動作成・命名・管理します。
  • 関連する「問題管理」ファイルを更新し、課題の履歴を管理します。
  • 氏名やメールアドレスなど、個人プロフィールデータは取得しません。

2. データの共有・開示

当アプリは、ユーザーデータを第三者に販売・貸与・共有・開示することは一切ありません。 すべてのデータは、ユーザーのGoogleアカウント内にのみ存在し、ユーザーの管理下にあります。

3. データの保存と削除

  • すべてのデータは、ユーザーのGoogleドライブ内に保存されます。
  • アプリは外部サーバーや独自のバックアップを一切保持しません。
  • ユーザー自身がファイルを削除することで、データを完全に消去できます。
  • アプリのアクセス権限は、以下のページからいつでも取り消すことができます：
    Google アカウントの権限管理

4. データ保護とセキュリティ

  • Google API を通じたすべてのデータアクセスは、安全かつユーザーの明示的な同意に基づいて行われます。
  • 最低限のアクセス権限のみを使用しており、以下のスコープに限定しています：
    • https://www.googleapis.com/auth/spreadsheets
    • https://www.googleapis.com/auth/gmail.send
    • https://www.googleapis.com/auth/drive
  • ユーザーデータはGoogleのセキュアな環境内でのみ処理され、外部に送信・保存されることはありません。
  • アプリはすべて、Google Apps Script の実行環境内で動作しています。

5. ユーザーの選択と責任

ユーザーは、どのファイルにアクセス・作成するかを完全に管理できます。 アプリの利用を中止したり、関連ファイルを削除したり、アクセス権限を取り消すこともいつでも可能です。 ドライブの共有設定を定期的に確認することを推奨します。
