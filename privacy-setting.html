Privacy Policy – Sakubun-kun Mondai Sakusei

Last updated: July 2, 2025

1. Overview

Sakubun-kun Mondai Sakusei is a tool that generates and manages question files for use with the Sakubun-kun SP/EX application. It utilizes Google Sheets and Google Drive for file creation, editing, and organization.

2. Data Collected and Purpose of Use

  • Google Sheets: Used to create and manage question sheets.
  • Google Drive: Used to store, organize, and replace files automatically.

3. Data Sharing and Disclosure

All files and data are stored within the user’s own Google Account. No user data is shared with or disclosed to any third parties.

4. Data Retention and Deletion

  • Files created or edited by this app remain in the user’s Google Drive.
  • No external servers are used. No data is transferred out of Google’s system.
  • Deleting a file through Drive will permanently delete the data.

5. Data Protection and Security

All access to user data is performed securely via Google APIs with the user’s explicit consent.

The following scopes are used (minimum required):

  • https://www.googleapis.com/auth/spreadsheets
  • https://www.googleapis.com/auth/drive

This application runs entirely within the secure Google Apps Script environment. No data is transmitted or stored outside Google services.
